### Basic Motion

| Key | Description |
|:--:|:--|
| <kbd>h</kbd> | Left |
| <kbd>j</kbd> | Down |
| <kbd>k</kbd> | Up |
| <kbd>l</kbd> | Right |

### Screen Motion
| Key | Description |
|:--:|:--|
| <kbd>H</kbd> | Top Screen |
| <kbd>M</kbd> | Middle Screen | 
| <kbd>L</kbd> | Botton Screen |
